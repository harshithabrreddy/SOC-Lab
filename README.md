# 🛡️ SOC Analyst Lab Portfolio

## 👋 Welcome

Welcome to my **SOC Analyst Lab Portfolio**.

I created this repository to document my journey of learning and practicing **Security Operations Center (SOC)** skills through realistic, hands-on investigations performed in a controlled lab environment.

Each project simulates a real-world incident and follows the same methodology used by SOC analysts:

* Receive an incident ticket
* Collect evidence
* Investigate the issue
* Identify the root cause
* Contain and recover the affected system
* Verify the resolution
* Document the findings
* Capture lessons learned

This repository demonstrates my ability to perform structured investigations, analyze system behavior, troubleshoot security and network issues, and produce professional documentation.

# 🔍 Investigation Workflow

Every lab follows a consistent investigation process:

1. Incident Ticket Received
2. Initial Assessment
3. Evidence Collection
4. System & Network Investigation
5. Log Analysis
6. Root Cause Analysis
7. Containment
8. Recovery
9. Verification
10. Incident Report
11. Lessons Learned


# 📁 Standard Folder Structure

Each investigation includes:

| File                        | Description                        |
| --------------------------- | ---------------------------------- |
| `01_Ticket_Details.txt`     | Original incident ticket           |
| `02_Evidence/`              | Screenshots, logs, command outputs |
| `03_Investigation_Notes.md` | Step-by-step investigation notes   |
| `04_Incident_Report.md`     | Professional incident report       |
